# Portfolio Profesional - Alexander Ocampo

Este repositorio contiene el código fuente de mi portafolio profesional, diseñado para mostrar mi trayectoria como desarrollador de software, mis proyectos destacados y mi stack tecnológico actual. La web está optimizada para ser servida a través de GitHub Pages.

## 🚀 Sobre el Proyecto

El objetivo de este portafolio es centralizar mi identidad digital y ofrecer a reclutadores y colaboradores una visión técnica detallada de mis capacidades. Como estudiante de **Desarrollo de Aplicaciones Multiplataforma (DAM)** y con formación previa en **Ingeniería Industrial**, este sitio refleja mi enfoque analítico y mi pasión por el desarrollo frontend.

### Secciones principales:
- **Sobre mí:** Resumen de mi perfil híbrido entre ingeniería y desarrollo.
- **Proyectos:** Galería detallada con enlaces a repositorios y demostraciones en vivo.
- **Currículum:** Sección de visualización y descarga de mi CV actualizado.
- **Contacto:** Formulario funcional y enlaces directos a mis redes profesionales.

## 🛠️ Tecnologías Utilizadas

Para este proyecto se han priorizado tecnologías estándar de la web para garantizar velocidad, accesibilidad y un SEO básico eficiente:

- **HTML5:** Estructura semántica para mejorar la accesibilidad.
- **CSS3:** Diseño responsivo (Mobile First) con variables personalizadas y animaciones suaves.
- **JavaScript (Vanilla):** Gestión de eventos, scroll suave y validación de formularios sin dependencias externas.
- **Google Fonts:** Integración de la tipografía "Inter" para una lectura clara y moderna.

## 📂 Proyectos Destacados

En el portafolio se detallan proyectos como:
1. **Eventos Culturales Puertollano:** Aplicación en Vue.js para la gestión de eventos con persistencia de datos.
2. **Gestión ITV Concesionarios:** Herramienta de automatización de procesos con lógica de cálculo dinámica.
